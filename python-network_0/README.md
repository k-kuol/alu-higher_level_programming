# Python - Network #0

Bash scripts and Python files demonstrating HTTP fundamentals using cURL.

## Scripts

| File | Description |
|------|-------------|
| `0-body_size.sh` | Displays the size of the response body in bytes |
| `1-body.sh` | Displays body of a 200 response |
| `2-delete.sh` | Sends a DELETE request and displays the response body |
| `3-methods.sh` | Displays all HTTP methods accepted by the server |
| `4-header.sh` | Sends a GET request with custom header `X-HolbertonSchool-User-Id: 98` |
| `5-post_params.sh` | Sends a POST request with email and subject parameters |
