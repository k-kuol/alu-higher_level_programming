# Python - More Classes and Objects

This project explores advanced Object-Oriented Programming concepts in Python, including class attributes, static methods, class methods, and special methods.

## Tasks

- 0-rectangle.py: Empty Rectangle class
- 1-rectangle.py: Rectangle with width and height properties
- 2-rectangle.py: Rectangle with area and perimeter methods
- 3-rectangle.py: Rectangle with string representation (__str__)
- 4-rectangle.py: Rectangle with eval representation (__repr__)
- 5-rectangle.py: Rectangle with deletion message (__del__)
- 6-rectangle.py: Rectangle with instance counter
- 7-rectangle.py: Rectangle with customizable print symbol
- 8-rectangle.py: Rectangle with static method for comparison
- 9-rectangle.py: Rectangle with class method to create squares

## Learning Objectives

- Understanding class vs instance attributes
- Working with static methods and class methods
- Implementing special methods (__str__, __repr__, __del__)
- Managing object lifecycle and instance counting
- Advanced OOP concepts and design patterns
