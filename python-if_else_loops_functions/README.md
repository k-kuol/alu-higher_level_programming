# Python - if/else, loops, functions

This project covers Python control flow, loops, and functions.

## Tasks

- 0-positive_or_negative.py: Check if a number is positive, negative, or zero
- 1-last_digit.py: Print the last digit of a number
- 2-print_alphabet.py: Print the alphabet in lowercase
- 3-print_alphabt.py: Print the alphabet except q and e
- 4-print_hexa.py: Print numbers in decimal and hexadecimal
- 5-print_comb2.py: Print numbers from 00 to 99
- 6-print_comb3.py: Print all combinations of two different digits
- 7-islower.py: Function to check if a character is lowercase
- 8-uppercase.py: Function to print a string in uppercase
- 9-print_last_digit.py: Function to print the last digit
- 10-add.py: Function to add two integers
- 11-pow.py: Function to compute power
- 12-fizzbuzz.py: FizzBuzz implementation
