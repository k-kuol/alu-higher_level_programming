# SQL Introduction

This project introduces the basics of SQL and MySQL databases.

## Learning Objectives

- What is a database and a relational database
- What SQL stands for and what MySQL is
- How to create and delete a database
- How to create and alter tables
- How to SELECT, INSERT, UPDATE, and DELETE data
- How to use MySQL functions and subqueries

## Requirements

- All files executed on Ubuntu 20.04 LTS using MySQL 8.0
- All SQL keywords in uppercase
- All files end with a new line
- All queries preceded by a comment describing the task

## Tasks

| File | Description |
|------|-------------|

| `7-insert_value.sql` | Insert a row into first_table |
