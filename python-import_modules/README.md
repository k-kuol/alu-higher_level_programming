# Python - import & modules

This project covers Python modules, imports, and command line arguments.

## Tasks

- 0-add.py: Import and use a simple function
- 1-calculation.py: Import functions from calculator module
- 2-args.py: Print command line arguments
- 3-infinite_add.py: Add all command line arguments
- 4-hidden_discovery.py: Print names from compiled module
- 5-variable_load.py: Import and print a variable
