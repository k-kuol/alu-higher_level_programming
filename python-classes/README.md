# Python - Classes and Objects

This project introduces Object-Oriented Programming (OOP) concepts in Python, focusing on classes, objects, attributes, and methods.

## Tasks

- 0-square.py: Empty Square class
- 1-square.py: Square class with private size attribute
- 2-square.py: Square class with size validation
- 3-square.py: Square class with area calculation method
- 4-square.py: Square class with getter and setter properties
- 5-square.py: Square class with printing functionality
- 6-square.py: Square class with position attribute and advanced printing

## Learning Objectives

- Understanding OOP concepts in Python
- Working with classes and objects
- Using private attributes and properties
- Implementing getters and setters
- Data validation and error handling
- Method implementation and documentation
