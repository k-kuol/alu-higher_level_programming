# Python - More Data Structures: Set, Dictionary

This project covers Python advanced data structures including sets, dictionaries, lambda functions, and map/filter/reduce.

## Tasks

- 0-square_matrix_simple.py: Square all integers in a matrix
- 1-search_replace.py: Replace all occurrences of an element
- 2-uniq_add.py: Add all unique integers in a list
- 3-common_elements.py: Return common elements in two sets
- 4-only_diff_elements.py: Return elements present in only one set
- 5-number_keys.py: Return number of keys in a dictionary
- 6-print_sorted_dictionary.py: Print dictionary by ordered keys
- 7-update_dictionary.py: Replace or add key/value in dictionary
- 8-simple_delete.py: Delete a key in a dictionary
- 9-multiply_by_2.py: Return new dictionary with all values multiplied by 2
- 10-best_score.py: Return key with biggest integer value
- 11-multiply_list_map.py: Multiply list values using map
- 12-roman_to_int.py: Convert Roman numeral to integer
