# Python - Exceptions

This project covers Python exception handling, error management, and proper use of try/except blocks.

## Tasks

- 0-safe_print_list.py: Safe list printing with exception handling
- 1-safe_print_integer.py: Safe integer printing with format validation
- 2-safe_print_list_integers.py: Print only integers from a list safely
- 3-safe_print_division.py: Safe division with try/except/finally
- 4-list_division.py: Divide two lists element by element safely
- 5-raise_exception.py: Raise a TypeError exception
- 6-raise_exception_msg.py: Raise a NameError with custom message
